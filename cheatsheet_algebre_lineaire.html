Sous-espaces vectoriels & Applications linéaires
Prouver qu'un ensemble est un SEV
1
Vecteur nul — vérifier que 0 ∈ F.
2
Stabilité additive — si u, v ∈ F alors u+v ∈ F.
3
Stabilité scalaire — si u ∈ F et λ ∈ 𝕂 alors λu ∈ F.
💡 Raccourci : si F = ker(φ) pour une app. linéaire φ, c'est automatiquement un SEV. Si F = Vect{v₁,...,vₖ}, c'est aussi un SEV.
Prouver qu'une application est linéaire
1
Vérifier f(u + λv) = f(u) + λf(v) pour tous u, v et λ ∈ 𝕂.
2
Ou séparément : additivité f(u+v)=f(u)+f(v) + homogénéité f(λu)=λf(u).
Noyau
ker(f) = { x ∈ E | f(x) = 0 }
Image
Im(f) = { f(x) | x ∈ E } = f(E)
Théorème du rang (fondamental !)
dim(E) = dim(ker f) + dim(Im f)
dim(E) = dim(ker f) + rg(f)
Utilisation typique : si on connaît dim(Im f), on en déduit dim(ker f) = dim(E) − rg(f), et inversement.

Diagonalisation
Étapes pour diagonaliser A (n×n)
1
Polynôme caractéristique : calculer χ_A(λ) = det(A−λI)
2
Valeurs propres : résoudre χ_A(λ) = 0 → λ₁,...,λₖ avec multiplicités algébriques m₁,...,mₖ
3
Sous-espaces propres : pour chaque λᵢ, calculer Eᵢ = ker(A−λᵢI) et sa dimension dᵢ
4
Test : A diagonalisable ⟺ dᵢ = mᵢ pour tout i
5
Matrice P : colonnes = vecteurs propres (base de chaque Eᵢ)
6
Matrice D : diag(λ₁,...,λ₁,λ₂,...,λₙ) avec chaque λᵢ répété dᵢ fois
Critères de diagonalisabilité
Conditions SUFFISANTES (pas besoin de vérifier plus)
✓
A a n valeurs propres distinctes dans 𝕂 → toujours diagonalisable
✓
A est symétrique réelle → diagonalisable dans R (théorème spectral)
✓
A vérifie une relation P(A)=0 avec P scindé à racines simples
Condition NÉCESSAIRE ET SUFFISANTE
Pour tout λᵢ : dim Eλᵢ = multiplicité algébrique de λᵢ
Équivalent : dim(ker(A−λᵢI)) = ordre de λᵢ dans χ_A

Forme de Jordan & Matrices nilpotentes
Matrices nilpotentes

N est nilpotente d'ordre k si Nᵏ = 0 et N^(k-1) ≠ 0.

Décomposition A = λI + N (N nilpotente)
Aⁿ = (λI + N)ⁿ = Σ C(n,k) λⁿ⁻ᵏ Nᵏ
La somme s'arrête à k−1 (car Nᵏ=0). Si N²=0 : Aⁿ = λⁿI + nλⁿ⁻¹N
Vérifier la nilpotence
1
Calculer N², N³,... jusqu'à trouver Nᵏ = 0
2
Ou : toutes les VP de N sont nulles ⟺ N nilpotente
3
Ou : χ_N(λ) = (−1)ⁿλⁿ ⟺ N nilpotente
Blocs de Jordan

Un bloc de Jordan de taille r pour la valeur propre λ est :

Jᵣ(λ) = λ 1 0 ... 0 λ 1 ... ... λ 1 0 ... 0 λ
Matrice λI + N où N est le décalage (shift) nilpotent d'ordre r.
Puissance d'un bloc de Jordan
Jᵣ(λ)ⁿ = Σₖ C(n,k) λⁿ⁻ᵏ Nᵏ (N^r=0)
Entrée (i,j) de Jᵣ(λ)ⁿ : C(n, j-i)·λⁿ⁻⁽ʲ⁻ⁱ⁾ si j≥i, 0 sinon
Trouver une chaîne de Jordan

Pour une VP λ avec dim Eλ < m.alg., construire une chaîne :

1
Trouver u ∈ ker(A−λI) (vecteur propre)
2
Résoudre (A−λI)v = u → v est vecteur propre généralisé
3
Résoudre (A−λI)w = v → w est vecteur propre généralisé d'ordre 3
4
Continuer jusqu'à remplir le sous-espace caractéristique
f(u)=λu, f(v)=λv+u, f(w)=λw+v

Formules clés à mémoriser
Polynôme caractéristique
χ_A(λ) = det(A − λI)
Propriétés :
• χ_A(λ) est de degré n
• χ_A(A) = 0 (Cayley-Hamilton)
• Coefficient de λⁿ : (−1)ⁿ
• Terme constant : det(A)
• Coefficient de λⁿ⁻¹ : (−1)ⁿ⁻¹ tr(A)
Trace & Déterminant
tr(A) = Σλᵢ = somme des VP
det(A) = Πλᵢ = produit des VP

A inversible ⟺ det(A) ≠ 0 ⟺ 0 n'est pas VP de A

Si A = PDP⁻¹ : det(A)=det(D)=Πλᵢ
Changement de base
D = P⁻¹AP
A = PDP⁻¹
Aⁿ = PDⁿP⁻¹
f(Aⁿ) = P·f(D)·P⁻¹ (pour tout pol. f)
Inverse d'une matrice 2×2
Si A = [[a,b],[c,d]] :
A⁻¹ = (1/det(A)) · [[d,-b],[-c,a]]
det(A) = ad − bc
Pour 3×3 : méthode des cofacteurs ou Gauss-Jordan sur [A|I]
Binôme de matrices qui commutent
(A+B)ⁿ = Σ C(n,k) AⁿBᵏ si AB=BA
Cas λI + N : λI et N commutent toujours, donc :
(λI+N)ⁿ = Σₖ C(n,k)λⁿ⁻ᵏNᵏ
Attention : si AB ≠ BA, le binôme ne s'applique pas !
Suites récurrentes linéaires

Pour u_{n+2} = α·u_{n+1} + β·uₙ :

A = [[0,1],[-β, α]] (matrice compagnon)
Uₙ = Aⁿ·U₀ où Uₙ = (uₙ, u_{n+1})ᵀ
Les valeurs propres de A = racines de l'équation caractéristique r²−αr−β=0
Si r₁ ≠ r₂ : uₙ = A·r₁ⁿ + B·r₂ⁿ

1
Former la matrice (A − λI) pour la valeur propre λ
2

3

4
Exprimer toutes les variables en fonction des variables libres

Méthode : Prouver non-diagonalisabilité
1
Calculer χ_A(λ) et trouver une VP λ₀ de multiplicité m ≥ 2
2
Calculer le rang de (A−λ₀I)
3
dim Eλ₀ = n − rang(A−λ₀I)
4
Si dim Eλ₀ < m → A n'est pas diagonalisable
La matrice A est alors trigonalisable (forme de Jordan) mais pas diagonalisable.

Méthode : Montrer A inversible via une relation polynomiale

Si on connaît P(A) = 0 pour un polynôme P, chercher à isoler I :

Exemple : A² = A + 2I
⟹ A² − A = 2I
⟹ A(A − I) = 2I
⟹ A · ½(A−I) = I
⟹ A⁻¹ = ½(A − I) ✓
Principe général : si aₖAᵏ + ... + a₁A + a₀I = 0 avec a₀ ≠ 0, alors A est inversible et A⁻¹ = −(1/a₀)(aₖAᵏ⁻¹+...+a₁I)
